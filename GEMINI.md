This repository will contain the code for a website called "CateGurumi", which will be used by my mother to
show off her Amigurumi creations.
The stack will be the following:

- Backend: Laravel
- Frontend: React

Remember that we are developing on MacOs, therefore I have Herd installed if needed.

The system will need an Admin login for my mother to upload pictures with a title and a small description.
For normal users, there will need to be a beautiful, mobile-first interface to preview the images and seeing the
details in a modal when clicking on the image.

The frontend will need to cache the images so there's no need to download them every time.

The Laravel backend will need to serve the built React frontend through Blade.
